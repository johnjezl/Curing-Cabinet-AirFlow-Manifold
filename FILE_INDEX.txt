SOURCE CODE:
------------

manifold_design.py (21 KB)
  Main parametric design generator
  - Written in Python using CadQuery 2.4.0
  - Generates all monolithic STL files
  - Fully parametric (edit constants at top)
  - Includes:
    * Design parameters (all dimensions)
    * Helper functions (area calculations)
    * Component builders (tubes, sensor mount, fan mount)
    * Main manifold sections (base, transition, chamber, adapter)
    * Snap-fit helpers (male/female connectors)
    * O-ring groove generator
    * Export function (generates STL files)
  - Run: python manifold_design.py

manifold_design_split.py (11 KB)
  Split base generator
  - Imports parameters from manifold_design.py
  - Splits 510×510mm base into 3×3 grid
  - Adds bolt holes and alignment pins
  - Generates 9 separate STL files
  - Each section: 170×170mm (fits Ender 3 V3 KE)
  - Run: python manifold_design_split.py

requirements.txt (16 bytes)
  Python package dependencies:
  - cadquery==2.4.0
  Install: pip install -r requirements.txt

STL FILES - SPLIT BASE:
-----------------------

base_section_X_Y.stl (9 files, ~460 KB each)
  Individual base sections for split assembly
  - Dimensions: 170×170×43mm each
  - Fits Ender 3 V3 KE print bed perfectly
  - Each section contains:
    * Bottom plate (3mm thick)
    * Wall structure (40mm tall, 3mm thick)
    * Intake tube socket(s) where applicable
    * Bolt holes on edges (for joining sections)
    * Alignment pins (male) or holes (female)
    * Snap-fit connectors on top (for transition section)
    * O-ring groove on top edge

  Layout (top view):
    0,2   1,2   2,2    ← Back
    0,1   1,1   2,1    ← Middle
    0,0   1,0   2,0    ← Front

  Print time per section: ~5 hours
  Material per section: ~200g PETG
  Total print time (9×): ~45 hours
  Total material (9×): ~1.8 kg PETG

STL FILES - MONOLITHIC BASE:
----------------------------

manifold_base.stl (483 KB)
  Single-piece base (alternative to split base)
  - Dimensions: 510×510×43mm
  - Too large for Ender 3 V3 KE
  - MUST BE SPLIT IN SLICER (e.g., PrusaSlicer, Cura)
  - Contains all features of split base in one piece
  - Advantages:
    * Fewer parts to assemble
    * No bolt assembly needed
    * Potentially stronger (no joints)
  - Disadvantages:
    * Must split in slicer
    * Harder to print (very large)
    * Requires excellent bed adhesion

  Print time: ~50 hours (if printable)
  Material: ~1.8 kg PETG

STL FILES - COMMON PARTS:
-------------------------

manifold_transition.stl (42 KB)
  Tapered transition section
  - Dimensions: 510×510mm bottom → 48×48mm top × 150mm height
  - Gradually concentrates airflow
  - Smooth internal taper for laminar flow
  - Features:
    * Female snap-fit on bottom (mates with base)
    * Male snap-fit on top (mates with chamber)
    * O-ring groove on top edge
    * Hollow interior (3mm wall thickness)
  - Print orientation: Vertical (as designed)
  - Supports: May be needed depending on taper angle
  - Print time: ~12 hours
  - Material: ~400g PETG

manifold_sensor_chamber.stl (25 KB)
  Sensor mounting chamber
  - Dimensions: 48×48×80mm
  - Contains 42×42mm airflow passage
  - Features:
    * Cross-struts for sensor PCB mounting
    * 4× screw holes (4mm dia) for M3 screws
    * Centered in airflow path
    * Female snap-fit on bottom (mates with transition)
    * Male snap-fit on top (mates with adapter)
    * O-ring groove on top edge
  - Sensor PCB: 1"×1" (25.4mm square)
  - Mounting holes: 23.4mm spacing (corner holes)
  - Print orientation: Vertical
  - Supports: May be needed for struts
  - Print time: ~3 hours
  - Material: ~100g PETG

manifold_fan_adapter.stl (183 KB)
  Fan mounting adapter
  - Dimensions: 48×48mm bottom → 130×130mm top × 44mm height
  - Transitions from sensor chamber to fan mount
  - Features:
    * Female snap-fit on bottom (mates with chamber)
    * Fan mounting plate on top (4mm thick)
    * 4× mounting holes (4.5mm dia) for M4 fan screws
    * 105mm bolt circle (standard 120mm fan)
    * Center opening (115mm dia) for airflow
    * Smooth internal taper
  - Fan size: 120mm × 120mm standard PC fan
  - Print orientation: Vertical (may need supports)
  - Supports: Recommended for overhang
  - Print time: ~5 hours
  - Material: ~200g PETG

intake_tube.stl (74 KB) - PRINT 9 COPIES
  Individual intake tube
  - Dimensions: Ø38mm OD × 60mm length
  - Inner diameter: Ø35mm (for airflow)
  - Features:
    * 60mm length (fits 1.5-2.5 inch freezer top)
    * O-ring groove at top (for gasket)
    * Smooth inner bore
    * Slight clearance for base socket (38.2mm OD socket)
  - Insert from below freezer, through holes
  - Mounts in base sockets from below
  - Print orientation: Vertical (standing on end)
  - Supports: None needed
  - Print time per tube: ~2 hours
  - Material per tube: ~80g PETG
  - Total print time (9×): ~18 hours
  - Total material (9×): ~720g PETG

SUPPORT & MODIFICATIONS:

To regenerate STL files with custom parameters:
  1. Install Python 3.x
  2. pip install -r requirements.txt
  3. Edit manifold_design.py (parameters at top)
  4. Run: python manifold_design.py
  5. New STL files overwrite existing ones

Common modifications:
  - Change cabinet size: Edit CABINET_WIDTH, CABINET_DEPTH
  - Change fan size: Edit FAN_SIZE, FAN_MOUNT_HOLE_SPACING
  - Change tube count: Edit NUM_TUBES_X, NUM_TUBES_Y
  - Change multiplier: Edit related parameters, run verify_speed_multiplier()

See README.md → "Design Modifications" section for detailed instructions.
